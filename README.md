# Music Player

**Music Player** is simple application to share music in a common repository and choose your favorite songs.


Installation
-----------------
#### Clone project
```
git clone git@github.com:rorlika/music-player.git
cd music-player
```
#### Install dependencies
```
bundle install
bundle update
```
#### Run migrations
```
rake db:migrate
```
#### Run server
```
rails server
```
## Requirements 
* rails version >= 4.2.5
* ruby version >=  2.2.3
* postgresql
